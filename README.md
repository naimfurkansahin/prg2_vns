# VNS: Akıllı Ses Deşifre ve Analiz Sistemi

Bu proje, Programlama ve Uygulamaları - II dersi kapsamında geliştirilen, yerel STT (Faster-Whisper) ve bulut tabanlı LLM (Llama 3) teknolojilerini birleştiren bir ses analiz aracıdır.

## 🚀 Özellikler
- **Yerel Deşifre:** Faster-Whisper ile gizlilik odaklı ses-metin dönüşümü.
- **Akıllı Özetleme:** Groq API üzerinden Llama 3 ile anlamsal analiz.
- **Modern Arayüz:** CustomTkinter ile kullanıcı dostu tasarım.
- **Veri Güvenliği:** KVKK uyumlu işlem ve yerel veritabanı.

## 🛠️ Kurulum
1. `git clone https://github.com/kullanici_adin/prg2_vns.git`
2. `pip install -r requirements.txt`
3. `python main.py`
